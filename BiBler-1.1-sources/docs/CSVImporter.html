Class CSVImporter¶

class app.impex.CSVImporter(path, manager)[source]¶

Bases: app.impex.Importer

Import entries Entries from a CSV (tabs) file. The first row must consist of the field names for each column. This operation is meant to be the inverse for the CSVExporter.

Parameters:

  • manager (list of app.manager.ReferenceManager) – The reference manager that will hold the entry list.
