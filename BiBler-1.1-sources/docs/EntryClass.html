Class Entry¶

class app.entry.Entry[source]¶

Bases: object

The abstract class representing entries. Every entry has an _id, a key, and dictionaries for requiredFields, optionalFields, and additionalFields as defined in BibTeX. importantFields are usually required from the optional ones, but if missing the entry is still valid.
